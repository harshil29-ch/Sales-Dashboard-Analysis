 Sales Dashboard Analysis

 Project Overview

This project presents an interactive **Sales Dashboard** built using Power BI to analyze business performance across different regions, categories, and time periods.

 Key Metrics

* Total Sales: 341K+
* Total Quantity: 5239
* Total Profit: 27K+

 Features

*  Monthly Sales & Profit Trends
*  Sales by Region (Map Visualization)
*  Category & Sub-Category Analysis
*  Shipping Mode Insights
*  Payment Mode Distribution

 Tools Used

* Power BI
* Excel (Data Cleaning)
* Data Visualization Techniques

 Insights

* Office Supplies generate highest revenue
* COD is the most used payment method
* Standard Class shipping dominates
* Sales peak during mid-year months

 How to Use

1. Download `.pbix` file
2. Open in Power BI Desktop
3. Interact with filters (Region, Year)
